CheekyCMS

CheekyCMS is a web editing interface for Content Management Systems. CheekyCMS can be adapted to work on almost any content management backend.

The user interface philosophy behind CheekyCMS is simple: all content that you are allowed to change becomes editable, right there on the page youâ€™re reading. Any modifications you make are retained in your browser and can be sent back to the CMS with a push of a button.
