CheekyCMS

CheekyCMS is a web editing interface for Content Management Systems. CheekyCMS can work with almost any content management backend.

The user interface philosophy behind CheekyCMS is simple: all content that you are allowed to change becomes editable, right there on the page youâ€™re reading. Any modifications you make are retained in your browser and can be sent back to the CMS with a push of a button.

Try it now

Edit in place

Edit your content right where it is shown, exactly the way it is shown with CheekyCMS.

No need to fuss around with admin interfaces or to second-guess how your finalized content will look on a page.

Lightweight and fast

CheekyCMS is a small JavaScript library that can be added to your site in minutes.

Once installed, authorized users need only navigate to the page to edit. When changes are made in the page, there is no need to learn or load a separate content management interface.

Works with any CMS

CheekyCMS is been written in JavaScript so that it can be integrated as the web editing interface on top of any Content Management System.
