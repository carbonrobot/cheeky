Cheeky
-----------------

[![Build Status](https://travis-ci.org/cheekycms/cheeky.svg?branch=master)](https://travis-ci.org/cheekycms/cheeky)

Cheeky is an ultralight CMS running over Mongodb. https://mongolab.com/databases/cheeky-cms

## Development

```
$ npm install
$ bower install
$ gulp serve
```
